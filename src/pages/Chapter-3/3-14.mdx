# śānta-udita-avyapadeśya-dharma-anupātī dharmī

#### The substratum is following the past, uprisen and undetermined essential qualities

<br />

**śānta** - quiescent

**udita** - uprisen

**avyapadeśya** - undetermined

**dharma** - essential qualities; highest duty

**anupātin** - following

**dharmīn** - substratum
