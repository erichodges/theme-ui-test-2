### [<](/Chapter-3/3-2) 3-3 &nbsp[>](/Chapter-3/3-4/)

# tat-eva-artha-mātra-nirbhāsaṃ svarūpa-śūnyam-iva samādhiḥ

#### As if empty of one’s own character, the purpose alone shines forth; that indeed is integration

<br />

**tat** - that, this

**eva** - thus

**artha** - purpose, aim; meaning; thing, object

**mātrā** - only; measure, quantity, sum, size, duration

**nirbhāsā** - shines forth

**svarūpa** - own character

**śūnya** - empty

**iva** - as if

**samādhi** - putting together; communion; combining; integration

<br />

#### Commentary by TKV Desikachar:

“Prajña comes only in Samādhi.”

<br />

#### Commentary by Paul Harvey:

“Dhāraṇā is the process of ‘holding onto’ the object.
Dhyānā is the process of ‘linking with’ the object.
Samādhi is the process of ‘integration into’ the object.”
Reflections on Yoga Sūtra Chapter 3 verses 1-3
